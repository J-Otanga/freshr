Privacy Policy

Last Updated: June 2026

Welcome to Freshr ("we," "our," or "us"). We are strongly committed to protecting your privacy and making sure you understand exactly how your data is handled when using our automated AI job-matching ecosystem (Freshr.co.ke).

1. Data We Access & Collect

To provide high-accuracy AI matching services, we handle the following processing parameters:

  • Account Credentials: Basic markers including your email, profile username parameters, and login structures.
  • Connected Repository Metadata: Public GitHub portfolio analysis files, code block architectures, language usage arrays, and overall contribution volumes.
  • Optimized Career Assets: Generated structural data mapping configurations and CV profiles stored within workspace cloud records.

2. How Information is Processed

We process information exclusively to match your technical profile directly with open opportunities. This includes parsing your natural programming workflows, running framework comparison models, contextually tailoring adaptive text strings for auto-generated CV templates, and keeping you informed via secure update logs.

3. Third-Party Governance & Protection

Freshr does not participate in any direct commercial data brokers network or sell user metrics. Information fields only move across processing targets under strict operational triggers:

  • Targeted Employer Portals: When you execute a direct profile application submission command.
  • Secure Artificial Intelligence Processors: Secure API data transit arrays utilizing private enterprise security channels to extract technical summary blocks from unstructured code sets.

4. User Autonomy & Removal Requests

You have full, unhindered control over your linked account profiles. You can revoke technical authorization links or erase account instances immediately via workspace controls. Triggering account termination instantly queues your parsed developer datasets for permanent structural purging from system-wide storage targets.

5. Inquiries & Administrative Support

For official inquiries regarding structural personal privacy layers or execution frameworks under standard regulatory laws, contact our admin team via admin@freshr.co.ke.
